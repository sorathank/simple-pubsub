
# Simple Pub-sub

A simple demostration of Publisher - Subscriber.



## Installation

Install my-project with npm

```bash
  yarn add ts-node -g
  yarn
```
    
## Run Locally

Clone the project

```bash
  git clone https://github.com/sorathank/simple-pubsub
```

Go to the project directory

```bash
  cd simple-pubsub
```

Install dependencies

```bash
  yarn add ts-node -g
  yarn
```

Run script

```bash
  yarn start
```
